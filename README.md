# مشروع تخرج التسويق الرقمي - تطبيق خطة نجاة

### 1- لينك جيت هاب
* **الرابط:** [https://github.com/seifesraa96-dev/digital-marketing-graduation-project](https://github.com/seifesraa96-dev/digital-marketing-graduation-project)

### 3- اسم المشروع
* **تطبيق خطة نجاة (Kheta Najah App)**

### 4- وصف المشروع في جملة
* تطبيق إسلامي شامل لتنظيم العبادات اليومية من أوراد وأذكار وختم للقرآن الكريم، مصمم لمساعدة المسلم على استغلال شهر رمضان المبارك وفق خطة عملية مدروسة.

### 6- عمل شيت فيه مهام لكل تيم مشروع (توزيع المهام)

| التخصص | المهام المسندة |
| :--- | :--- |
| **مسؤول الاستراتيجية** | رسم خارطة الطريق للحملة التسويقية وتحديد الأهداف قبل وأثناء شهر رمضان. |
| **كاتب المحتوى** | صياغة المنشورات الدينية، المحتوى التحفيزي، وشرح مميزات التطبيق بأسلوب جذاب. |
| **المصمم الجرافيكي** | تصميم الهوية البصرية (Logo)، صور المنشورات، وتصاميم الإعلانات الممولة. |
| **أخصائي SEO / ASO** | تحسين محركات البحث داخل متجر التطبيقات لضمان ظهور التطبيق في النتائج الأولى. |
| **مدير الإعلانات** | إطلاق وإدارة الحملات الإعلانية المدفوعة على منصات التواصل الاجتماعي (Facebook, Instagram). |

### 8- فكرة المشروع
* **الفكرة:** تطبيق "خطة نجاة" هو مساعد رقمي ذكي يهدف إلى حل مشكلة تشتت الصائم وضياع الوقت في رمضان. يوفر التطبيق جداول عبادة مرنة تناسب مختلف الفئات، مع نظام تنبيهات للأذكار والأوراد، وعداد لمتابعة ورد ختم القرآن، لضمان تجربة إيمانية منظمة ومثمرة.
